About Project

  • "CODE EVEN" LMS사이트 - 프론트 (일반 유저용) 페이지 제작 프로젝트
  • 프로젝트 목표 : LMS 사이트 프론트 페이지 제작
  • 프로젝트 세부목표 : PHP로 DB를 활용한 프로젝트 제작하기, 팀원별 파트를 맡아 파트별 기획 구현 및 회의 진행
  • 주요 기능 및 설명:

    백엔드 페이지와의 기본 CRUD기능 및 PHP와 MySQL을 활용한 DB 연동

  • 프로젝트 소개: 피그마 슬라이드 덱
  • 프로젝트 깃허브: 깃허브
  • 배포: 깃허브, 닷홈, Vercel
